# <img src="./icon/logo.png" width="50px" /> Startpage - Trang chủ trình duyệt cá nhân hóa của bạn

Startpage là một tiện ích mở rộng Chrome thay thế trang tab mới bằng một giao diện gọn gàng, đầy đủ tính năng và đẹp mắt. Toàn bộ dữ liệu được lưu cục bộ — không tài khoản, không theo dõi.

### Hiệu ứng hình ảnh (48 hoạt ảnh + Bộ tạo sóng SVG)

Chọn từ 48 hiệu ứng canvas, cùng với Bộ tạo sóng SVG, với bộ chọn màu riêng cho các hiệu ứng được hỗ trợ:

| :--- | :--- | :--- | :--- |
| • Mưa / Mưa HD | • Mạng lưới (Network) | • Tuyết rơi / HD | • Ánh sáng Gradient |
| • Đom đóm / HD | • Mưa Matrix | • Sóng cực quang | • Bong bóng |
| • Mưa sao băng | • Hacker | • Cực quang (Northern Lights) | • Sóng ký tự |

| • Cột ánh sáng | • Pixel chạy | • Lá xanh | • Họa tiết chéo |

### Bộ tạo sóng SVG

- Tùy chỉnh sóng nền: số đường, biên độ X/Y, độ lệch, góc, độ mượt, chế độ tô màu
- Gradient màu HSL (bắt đầu → kết thúc)
- Lưu sóng tùy chỉnh vào thư viện cá nhân

### Cá nhân hóa nâng cao

- **Tiêu đề nổi tùy chỉnh (Custom Title)** — thêm văn bản của riêng bạn, di chuyển tự do (Free Move) và tùy chỉnh mọi thứ: phông chữ, kích thước, khoảng cách chữ, đổ bóng (X/Y/Blur/Màu) và viền.
- **Hệ thống đa màu sắc (Multi-color)** — tự động thay đổi màu nhấn giao diện với nhiều chế độ (Gradient, Chuyển màu theo thời gian, hoặc Đồng bộ với hình nền).
- **Menu truy cập nhanh (LCP)** — một menu nhỏ giúp bật/tắt các thành phần và điều chỉnh bố cục (Độ rộng, Vị trí) trực tiếp mà không cần mở cài đặt đầy đủ.
- **Giao diện Glassmorphism** — phong cách kính mờ chất lượng cao, đồng nhất trên tất cả các tiện ích và hộp thoại.

### Cài đặt & Cá nhân hóa

- **Chọn phông chữ** với hỗ trợ tải font tùy chỉnh
- **Màu nhấn** áp dụng trên toàn bộ giao diện
- **Tiêu đề trang** tùy chỉnh
- **Biểu tượng tab** tùy chỉnh — đặt 2 ký tự hoặc 1 emoji làm favicon tab trình duyệt
- **Ngôn ngữ**: Tiếng Anh và Tiếng Việt (i18n)
- **Bật/tắt từng thành phần**: Đồng hồ, Tìm kiếm, Dấu trang, Nhóm dấu trang, Công việc, Hẹn giờ, Notepad, Lịch, Âm lịch, Trình phát nhạc, Truy cập nhanh
- **Chế độ Ghost** cho các điều khiển bên
- **Sao lưu Đám mây (Cloud Sync)** — sao lưu và khôi phục cài đặt (không bao gồm ảnh/video) qua tài khoản Chrome của bạn
- **Quản lý Phông chữ** — sử dụng font có sẵn, Google Fonts, hoặc **duyệt và sử dụng các font đã cài đặt trên máy tính của bạn** (Local Font Access API)
- **Bố cục Dấu trang** — chọn giữa dạng Lưới mặc định, Thanh bên, và nhiều chế độ Thanh ngang (**Dưới giữa, Trên giữa, Dưới trái**)
- **Xuất / Nhập** cài đặt dạng file JSON
- Đặt lại tất cả cài đặt về mặc định
- Lưu vị trí các thành phần có thể kéo thả

## Cài đặt

1. Tải hoặc clone repository này.
2. Mở `chrome://extensions/` trong Chrome.
3. Bật **Chế độ nhà phát triển** (góc trên bên phải).
4. Nhấn **Tải tiện ích chưa đóng gói** và chọn thư mục dự án.
5. Mở tab mới — tận hưởng Startpage của bạn!

## Công nghệ

- **Vanilla JavaScript** — không framework, không bước build
- Chrome Extension **Manifest V3**
- Canvas API cho hoạt ảnh và visualizer
- LocalStorage & IndexedDB cho dữ liệu và cấu hình
- Chrome Bookmarks API để nhập bookmark
- Chrome Media Session API để phát hiện nhạc đang phát
- Chrome Storage Sync để đồng bộ hóa dữ liệu giữa các thiết bị
- Local Font Access API để tích hợp phông chữ hệ thống
