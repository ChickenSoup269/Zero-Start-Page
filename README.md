# Hướng dẫn cài đặt Startpage Extension

1.  Mở trình duyệt Chrome hoặc một trình duyệt dựa trên Chromium khác.
2.  Đi tới `chrome://extensions`.
3.  Bật "Chế độ dành cho nhà phát triển" (Developer mode) ở góc trên bên phải.
4.  Nhấp vào "Tải tiện ích đã giải nén" (Load unpacked).
5.  Chọn thư mục chứa extension này (thư mục `Startpage`).
6.  Mở một tab mới để xem trang bắt đầu mới của bạn!
